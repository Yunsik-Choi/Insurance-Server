# Insurance Server

# 목적

- 기존 보험사 운영 시스템을 Cloud 환경으로 Migration

# 사용 기술

- Spring Boot, Spring MVC, Spring Data JPA, Spring Cloud

# 기술적인 집중 요소

- Spring Cloud를 활용해 대용량 트래픽에도 유연하게 작동할 수 있는 서버 개발
